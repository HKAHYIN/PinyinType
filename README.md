# PinyinType

[![Website](https://img.shields.io/badge/Website-Visit%20PinyinType-blue?style=flat-square)](https://hkahyin.github.io/PinyinType/)

PinyinType is a tool/project for typing or converting Chinese Pinyin.  
This repository contains the source code and configuration for the project.

## Features

- Convert Chinese characters to Pinyin
- Support for tone marks
- Easy integration and usage
